Der eingegebene Modulcode wird bereits verwendet!

Sie möchten für das Modul {{prototype.name.title}} den Code {{prototype.name.code}} festlegen. Dieser Code wird momentan allerdings vom Modul {{conflict.name.title}} belegt. Sie können den Konflikt auf folgende Arten auflösen:

  1. Einfügen (Noch nicht implementiert): Das neue Modul wird vor dem alten Modul in die Liste eingefügt und alle nachfolgenden Module in dieser Gruppe rücken ein Feld nach hinten
  2. Anfügen: Ihr festgelegter Code wird verworfen und das neue Modul stattdessen an den Schluss dieser Kategorie angefügt. Der Code des neuen Moduls wäre dann {{prototype.appendCode}}. Falls das Modul bereits besteht, wird es versetzt
  3. Verwerfen: Das geänderte bzw. erzeugte Modul wird nicht in den Datensatz aufgenommen
  4. Zurück: Sie kehren zurück zur Modulbearbeitung und können den Code selbstständig neu setzen
